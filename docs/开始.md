# 开始

EUI-TEST 当前是一套轻量 OpenGL UI 实验工程。推荐使用 DSL 写页面和组件，由 Runtime 统一处理布局、交互、动画、脏区渲染和 primitive 生命周期。

## 工程结构

```text
main.cpp                程序入口、窗口、主循环
app/app.h              app 生命周期接口
app/dsl_app.h          DSL app 适配层
app/demo.cpp           简单 DSL 示例
app/gallery.cpp        Gallery 示例
core/layout.h          布局系统
core/primitive.h       圆角矩形底层图元
core/text.h/.cpp       文本底层图元
core/event.h           pointer 事件和交互状态
core/animation.h       transition / easing / animated value
core/dsl.h             DSL 元素和 builder
core/dsl_runtime.h     DSL Runtime、动画推进、脏区渲染
components/*.h         组件层 DSL helper
assets/                字体资源
docs/                  文档
```

## 推荐写页面方式

新页面优先使用 `app/dsl_app.h`：

```cpp
#include "app/dsl_app.h"
#include "components/components.h"

namespace app {

const DslAppConfig& dslAppConfig() {
    static const DslAppConfig config = {
        "My App",
        "my-app",
        {0.07f, 0.08f, 0.10f, 1.0f},
        1280,
        800
    };
    return config;
}

void compose(core::dsl::Ui& ui, const core::dsl::Screen& screen) {
    ui.stack("root")
        .size(screen.width, screen.height)
        .align(core::Align::CENTER, core::Align::CENTER)
        .content([&] {
            components::button(ui, "primary")
                .size(220.0f, 58.0f)
                .text("Click")
                .onClick([] {
                    // update state
                })
                .build();
        });
}

} // namespace app
```

## 推荐写组件方式

组件应该只组合 DSL：

```cpp
components::button(ui, "save")
    .size(180.0f, 54.0f)
    .text("Save")
    .onClick(...)
    .build();
```

不要在组件里直接创建 `RoundedRectPrimitive` 或 `TextPrimitive`。这样组件才能自动获得：

- Runtime 生命周期管理
- hover / press
- transition 动画
- dirty rect 渲染
- DPI 处理

## 构建

工程使用 CMake：

```powershell
cmake --build build --config Debug
cmake --build build --config Release
```

`CMakeLists.txt` 会为 `app/*.cpp` 中每个 cpp 生成一个可执行程序，例如：

- `demo.exe`
- `gallery.exe`

如果 Release 链接时报 `gallery.exe` 或 `glfw3.dll` 被占用，通常是程序窗口仍在运行，关闭后再构建即可。

## 运行示例

- `demo`：简单卡片和按钮。
- `gallery`：完整 Gallery，包含 Controls / Text / Animation / Settings / About。

## 当前开发约定

- 页面使用 DSL。
- 组件使用 DSL。
- app 状态可以用简单静态变量保存。
- 点击回调修改状态后 Runtime 会请求重新 compose。
- hover / press / transition 不需要手动重绘。
- 不要手动管理 primitive 生命周期。
